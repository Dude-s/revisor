Privacidad y recomendaciones

El PDF se procesa dentro del navegador de la persona que usa la página. No se sube a GitHub, no se manda a un servidor y no se guarda en una base de datos.
Recomendado: PDFs menores a 100 MB y con texto seleccionable. Si el PDF es escaneado como imagen, esta versión no podrá leerlo correctamente.
La detección de espaciado visual es aproximada. Para resultados importantes, revisa manualmente los casos marcados como “Revisar”.
